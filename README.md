# FUTURE_DS_01
# 📊 Revenue Performance & Market Insights Dashboard

## 📌 Project Overview
This project is part of the **Future Interns Data Science & Analytics Track (DS)**.  
The objective was to analyze raw transactional sales data and develop an interactive dashboard that provides meaningful business insights.

The project demonstrates an **end-to-end data analysis workflow**, from data cleaning in Excel to dashboard development in Power BI.

---

## 🎯 Objective
- Clean and prepare raw sales data for analysis  
- Identify key revenue drivers and sales trends  
- Analyze product performance and geographic distribution  
- Build an interactive dashboard to support decision-making  

---

## 🧹 Data Cleaning (Microsoft Excel)
Data cleaning was performed using Excel to ensure accuracy and consistency before analysis.

### Key Steps:
- Removed duplicate records  
- Handled missing values  
- Standardized date formats  
- Converted text-based numbers into numeric format  
- Cleaned inconsistent product descriptions  
- Identified and handled **negative quantities (returns)**  

---

## 📊 Dashboard Development (Power BI)

An interactive dashboard was built in Power BI to visualize and explore the data.

### Key Features:
- **KPIs:** Total Revenue, Units Sold, Number of Products, Countries  
- **Top Products Analysis:** Identifies highest revenue-generating products  
- **Country Analysis:** Shows sales distribution across different regions  
- **Monthly Trends:** Highlights seasonality and sales patterns  

---

## 📈 Key Insights
- Total revenue reached **10.25M**, with over **6M units sold**  
- A small number of products contribute significantly to total revenue  
- The **United Kingdom dominates sales**, indicating market concentration  
- Sales show strong **seasonal trends**, peaking in November and December
